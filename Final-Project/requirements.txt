#
# 2. Install Python packages:
#    pip install -r requirements.txt
#
# 3. Install Ollama for local LLM:
#    Download from: https://ollama.com
#    Then pull a model:
#      ollama pull llama3.2          (general purpose, recommended)
#      ollama pull dolphin-mistral   (uncensored, no restrictions)
#      ollama pull mistral           (fast 7B model)
#
# 4. Run the application:
#    python main.py
#

# ─────────────────────────────────────────────────────────────
#
# llama3.2        Meta's LLaMA 3.2 - best for financial analysis

#                 runs 100% locally, no data leaves your machine
# mistral         Mistral 7B - fast responses, good quality
# qwen2.5         Alibaba Qwen - strong multilingual support
#
